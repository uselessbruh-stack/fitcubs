# Fit Cubs Kids Mini Marathon - Event Registration System

## Overview

This is a comprehensive event registration web application built for the Fit Cubs Kids Mini Marathon. The system provided complete event management including participant registration, real-time tracking, and administrative oversight for the successful execution of the event.

## Features Delivered

- **User Registration**: Seamless participant registration with detailed event information collection
- **Admin Dashboard**: Comprehensive administrative portal to monitor registrations and event statistics
- **Real-time Analytics**: Dashboard to track registration counts and participant data
- **Responsive Design**: Fully optimized for desktop and mobile devices for accessibility
- **Secure Backend**: Robust backend API for secure data management and handling

## System Architecture

The application was built with a modern three-tier architecture:

- **Frontend**: User-facing registration interface and event information portal
- **Backend**: RESTful API for handling registrations and data management
- **Admin Portal**: Dedicated administrative dashboard for event oversight and analytics

## Technology Stack

- **Frontend**: React/JavaScript (Vite)
- **Backend**: Node.js/Express
- **Admin Portal**: React-based dashboard
- **Database**: Persistent data storage for registrations and analytics
